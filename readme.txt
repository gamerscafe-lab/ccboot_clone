# CCBoot Clone - Starter (MVP)
خطوات إعداد وتشغيل النسخة الأولى على Windows، مع افتراض أنك تريد وضعها على K:\ccboot_clone

## 1) فك الضغط في:
K:\ccboot_clone

## 2) تشغيل المشروع لأول مرة
- افتح المجلد: K:\ccboot_clone
- دوبل كليك على: run.bat
  > السكربت هيعمل:
  > - إنشاء بيئة افتراضية (.venv)
  > - تثبيت المتطلبات
  > - تشغيل السيرفر على العنوان http://127.0.0.1:8000

لو الـ "py" launcher مش موجود، السكربت بيحاول استخدام "python" تلقائيًا.

## 3) الاستخدام
- افتح المتصفح: http://127.0.0.1:8000
- أضف الأجهزة (Name + MAC).
- البيانات تتخزن داخل db.sqlite في نفس المجلد.

## ملاحظات
- هذه النسخة مجرد لوحة تحكم لقاعدة البيانات كبداية.
- لاحقًا هنضيف: ربط MAC بـ iSCSI، توليد ملفات iPXE، سكربتات PowerShell لإنشاء VHD تفاضلي لكل جهاز، وزر Reset.
